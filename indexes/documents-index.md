## Update Rules

- Every `documents/**/*.md` page must be listed here.
- Prefer stable `document_id` over display title for programmatic lookup.
- `Source` should point to the document page or source text, not just raw PDF.
- If a document is superseded, keep the row and update `Status` / `Review Status`.
