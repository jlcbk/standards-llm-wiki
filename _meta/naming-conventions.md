# Naming Conventions

## Standard IDs

- 正文：`GB 7258-2017`
- 文件名：`gb-7258-2017.md`
- 标准族目录：`gb-7258/`

## Directory Rules

- 原始 PDF：`raw/standards/gb-7258-2017.pdf`
- 转写文本：`sources/standards/gb-7258-2017.md`
- 标准页面：`standards/gb/gb-7258/gb-7258-2017.md`
- 当前版本入口：`standards/gb/gb-7258/current.md`
- 版本变化：`standards/gb/gb-7258/changelog.md`
- 条款页面：`clauses/gb-7258-2017/clause-11-6-seat.md`

## Topic Slugs

- 座椅：`seats`
- 安全带：`seat-belts`
- 制动：`braking-system`
- 灯光：`lighting-system`
- 外廓尺寸：`vehicle-dimensions`
- 校车：`school-bus`
- 新能源汽车：`new-energy-vehicle`
