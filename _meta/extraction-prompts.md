# Extraction Prompts

## Standard Page Extraction Prompt

请从以下标准原文中抽取结构化信息，生成适合 AI 知识库使用的 Markdown 页面。要求：

1. 保留标准号、标准名称、发布日期、实施日期、替代关系、适用范围。
2. 总结章节结构。
3. 抽取关键要求，按主题分类。
4. 识别涉及的车辆类型、部件系统、检测/合规场景。
5. 识别规范性引用文件。
6. 标记不确定或需要人工核查的信息。
7. 不要编造原文没有的信息。

## Topic Page Extraction Prompt

请从多个标准页面和条款页面中整理某一主题的跨标准知识页。要求：

1. 定义主题边界。
2. 列出直接相关标准。
3. 列出关键条款。
4. 按车辆类型或适用对象分类。
5. 标记高优先级标准和低优先级关联标准。
6. 给出常见问答提示。
