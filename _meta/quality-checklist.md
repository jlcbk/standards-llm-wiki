# Quality Checklist

## 入库前检查

- [ ] 原始文件已放入 `raw/`。
- [ ] 原文转写已放入 `sources/`。
- [ ] 标准主页面已创建或更新。
- [ ] 发布日期、实施日期、替代关系已记录，未知则标记 `unknown`。
- [ ] 关键主题已加入 `topics/` 或 `indexes/topic-to-standards-index.md`。
- [ ] 重要条款已创建 `clauses/` 页面。
- [ ] 版本变化已加入 `changelog.md` 或 `comparisons/`。
- [ ] `index.md` 已更新。
- [ ] `log.md` 已追加记录。

## 回答质量检查

- [ ] 是否能追溯到具体标准？
- [ ] 是否能追溯到具体条款？
- [ ] 是否能追溯到原始文件？
- [ ] 是否区分现行、废止、征求意见、未知？
- [ ] 是否明确说明不确定信息？
